## 🛠 変更内容
### ✨ UI刷新・エフェクト追加
- **フラッシュメッセージ (God Mode Neon)**:
    - 通知を画面中央上部の「Dynamic Island」スタイルに変更。
    - ダークモード時にネオンのように発光するエフェクトを追加しました。
- **フッターの新設**:
    - ページ下部にフッターを追加し、サイト全体のナビゲーションを整理。
    - FAB内にあった「利用規約」「プライバシーポリシー」をフッターへ移動させました。
### 🐛 不具合修正・レイアウト調整
- **投稿表示**:
    - 長文投稿がカード枠からはみ出す問題を `word-break: break-all` 等で修正。
    - スタンプピッカー（ポップオーバー）がカードの `overflow: hidden` で見切れる問題を解消。
    - スタンプピッカー背景の不透明度を上げ（95%）、視認性を向上させました。
- **SNSシェア**:
    - シェア文言内の改行コードをスペースに置換し、URL生成時の不具合を防止。
    - シェア定型文から固定文言を削除し、ユーザー入力を優先するように調整。
- **OGPプレビュー**:
    - モーダル閉じるボタンの反応エリア（背景クリック）を修正し、確実に閉じられるように改善。
    - タイトル文字と閉じるボタンが重ならないようパディングを調整。
    - 画像読み込み完了時にローディング表示が消えない不具合を修正。
### 🖱️ 操作性改善（PC向け）
- **ドラッグスクロール**:
    - スタンプ一覧などの横スクロール領域において、マウスドラッグでのスクロール操作を可能にしました ([scroll_drag_controller.js]。
### ⚙️ 開発環境・CI
- **テストスクリプト修正**: [bin/test_all.sh]を直列実行に変更し、テストの安定性を向上。
- **実行権限の修正**: GitHub Actionsでのエラー（Permission denied）を防ぐため、`bin/` 配下のスクリプトに実行権限を付与。
